## Prompt-001 | 2026-05-20 | 生成完整基础应用
**输入给AI的提示词**：
帮我创建一个AI选课决策助手Web应用，名字叫CourseMate，严格按照以下要求实现：

一、核心功能（必须全部实现）
1. 首页：大标题"AI选课决策助手 CourseMate"，输入专业下拉框（计算机、数学、英语、物理）和多选兴趣标签（轻松水课、高分友好、干货满满、老师nice、期末论文），点击生成按钮后输出至少2套课程推荐方案，每套包含3-5门课，显示课程名、学分、上课时间和详细推荐理由
2. 课程浏览页：可以按学院、课程类型、上课时间筛选课程，每个课程是卡片样式，点击进入详情页
3. 课程详情页：显示课程基本信息，AI生成的一句话评价摘要，用户评价列表，以及评价提交表单（1-5星评分、文字输入、截图上传）
4. 我的课表页：标准大学周课表（周一到周五每天8节课），支持拖拽课程到课表，自动检测时间冲突并高亮红色，支持导出课表为图片

二、技术要求
1. 纯前端实现，所有数据存在浏览器localStorage，不需要后端
2. 内置15门以上模拟课程数据，包含课程名、学院、学分、上课时间、课程类型
3. 评价截图自动转为Base64存储，限制单张图片小于1MB
4. 页面加载速度不超过3秒，界面美观，使用Tailwind CSS
5. 所有数据刷新页面不丢失

三、额外要求
1. 底部添加版权信息：© 2026 CourseMate | 蒋晨昊 230141918
2. 顶部有导航栏，可以在四个页面之间切换
3. 按钮点击有加载动画，操作有反馈提示
4. 预留OpenAI GPT-4o mini的API调用位置，方便后续替换
5. 如果API调用失败，自动显示预写的静态推荐数据和评价摘要

**AI完成的工作**：生成了包含4个页面的完整应用，实现了所有基础功能
